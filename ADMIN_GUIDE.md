# מדריך והגדרות לממשק הניהול (Admin Panel)

ממשק הניהול יושב תחת הנתיב `/admin` ומאפשר לנהל את המערכת והאופרציה של החנות.

## נתיבי מנהל (Frontend)
- **`/admin/login`** - מסך ההתחברות לאדמין. מתחבר במאחורי הקלעים ל-api שמאמת את פרטי הכניסה.
- **`/admin/dashboard`** - לוח הבקרה שיהווה את ליבת הניהול. נטען בנפרד טעינה מושהית (Lazy loaded) כדי לחסוך בביצועים כלליים של החנות למשתמשים רגילים.

## נתיבי מנהל (Backend / Vercel Functions)
תיקיית `api/admin/` כוללת קבצים כמו:
- `login.ts` (לאימות מנהל)
- `update-order.ts` (לניהול הזמנות אדמין)

## הרחבה צפויה לממשק הניהול מול Neon
לוח הבקרה מיועד לעבור הרחבות משמעותיות לצורך מעבר לניהול דינמי:
1. **לשונית "קטלוג":** הוספה, הסרה, ועריכה של קטגוריות ומוצרים (שם, סוג, מחיר, תמונה, סטטוס מלאי) ישירות למסד הנתונים Neon.
2. **לשונית "עיצוב ויזואלי":** שליטה על נראות האתר (צבע רקע הבאנר העליון, טקסט הבאנר המופיע, בחירת ותצוגת תמונת באנר מהמאגר).
המידע יוזן ישירות לטבלאות ייעודיות שיוקמו ויאופיינו בבסיס הנתונים PostgreSQL של הפרויקט.
